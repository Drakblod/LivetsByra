# Begravningsbyrå Hillbom - Website

En premium och vördnadsfull webbplats för Begravningsbyrå Hillbom, en ceremoniell byrå grundad i omtanke, respekt och närvaro.

## Funktioner
- Premium mörkt tema med gulddetaljer.
- Fixerad bakgrund med rogivande naturtema.
- Responsiv design anpassad för desktop och mobil.
- Interaktiva citat och scroll-animationer.

## Installation
Öppna `index.html` direkt i din webbläsare eller kör en lokal server:
```bash
npx http-server
```
